## рҹ§© What's Inside

KillTheHost is a bundle of two open-source, single-file Python tools designed to eliminate the gap between local development and live deployment.

| Tool | Version | Purpose |
|---|---|---|
| рҹҗҳ **PHP-MNGR** | `v2.4` | Local PHP project manager вҖ” start, stop, and monitor sites |
| рҹ—„пёҸ **DB-3NGIN3** | `v1.1` | Local database service manager вҖ” PostgreSQL, MySQL, Redis, MongoDB |

Together, they connect to your **Namecheap** domains and route traffic through **Cloudflare Tunnels** вҖ” putting your localhost on the public internet without a single line of server config.

Connect your **Namecheap** account and assign real domains to local projects вҖ” no manual DNS editing required. More registrar integrations are on the roadmap.

### вҳҒпёҸ Cloudflare Tunnel Integration  
Link a free Cloudflare account and expose local services to the public web securely. No port forwarding. No router config. No tunnel scripts.

### рҹҡҖ One-Click Public Access  
From localhost to a live URL in seconds вҖ” perfect for client previews, team demos, and real-world testing without a full deployment pipeline.

### рҹҗҳ PHP Project Control  
View runtime state, port, PHP version, and filesystem path for every local site. Start and stop projects from a clean interface instead of juggling terminal tabs.

### рҹ—„пёҸ Database Service Management

## рҹ“Ұ Installation

**1. Download the bundle**

```bash
# Download and unzip
curl -L https://killthehost.com/downloads/KillTheHost-v1.0.zip -o KillTheHost.zip
unzip KillTheHost.zip
cd KillTheHost
```

**2. Install dependencies**

```bash
pip install -r requirements.txt
```

**3. Run PHP-MNGR**

```bash
python php-mngr.py
```

**4. Run DB-3NGIN3**

```bash
python db-3ngin3.py
```

> Both tools can be run independently or together as part of the full KillTheHost workflow.

## рҹӣ пёҸ Usage

### Connecting a Domain

1. Open KillTheHost and navigate to **Domain Sync**
2. Enter your **Namecheap API key**
3. Select a domain and assign it to a local site
4. Done вҖ” no DNS panel needed

### Going Live with Cloudflare

1. Log in with your **Cloudflare account** (free tier works)
2. Select a local project or database service
3. Hit **Expose** вҖ” KillTheHost handles the tunnel setup
4. Your site is now reachable at your real domain
